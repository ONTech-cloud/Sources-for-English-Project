Notes on usage & citation

Below we list quick best-practice recommendations you can show to the teacher when citing images:

  • Always include the exact page URL you used (see each "Open source" button).
  • When an image is licensed (Shutterstock, Vecteezy with license), ensure you credit and obey the license terms.
  • If an image is used as a thumbnail, link back to the original so the teacher can verify source and license.
